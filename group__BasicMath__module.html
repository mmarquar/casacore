Detailed Description

Classes and global functions for basic math use. 

 @anchor BasicMath_module_anchor

See below for an overview of the classes in this module.

Synopsis

This module is a bag of related basic math classes and global functions.

The following functionality is available:

  • Templated functors that can be used with std::transform to apply functions like sin, near, sqrt, etc. to iterators on sequences like Array, Block, std::vector, etc.
  • Templated functions ConvertScalar to convert scalars from one type to another.
  • Class Functional to map a domain object into a range object.
  • Functions Math to interface to math.h and other scalar math functions.
  • Class Random to offer random number generators.
  • Prime numbers


Tip: You may want to look at the individual header files to see whether you might not prefer to include only the header files you really need; it may be more efficient to do so;
