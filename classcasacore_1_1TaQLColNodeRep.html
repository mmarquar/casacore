Detailed Description

Raw TaQL parse tree node defining a select column expression.

Intended use:

Internal

Review Status

Test programs:
tTaQLNode

Prerequisite

  • TaQLNodeRep

Synopsis

This class is a TaQLNodeRep holding a column expression in the column list of the select clause. A new column name and data type can be defined for the column (expression). The expression can be a wildcarded column name (a regex) preceeded by ~ or !~ (meaning include or exclude).
