Obtaining AIPS++

The released version of AIPS++ is available only as a binary distribution (but with source code) on CDROM. If you wish to receive a copy of the latest CDROM, please email aips2-request@nrao.edu with the following information: your name and mailing address, email address and preferred computer architecture. There is no cost.

AIPS++ supports use of Solaris on Sparc architectures, and Linux RedHat 6.*, and SuSE 6.2 on Intel architectures. When requesting a CD-ROM, please specify which version you would prefer. AIPS++ does not currently support Windows NT, Windows 95, Windows 98, or any Apple computers.
