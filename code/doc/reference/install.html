Obtaining AIPS++

The released version of AIPS++ is only available via CDROM. If you wish to receive a copy of the latest CDROM, please email aips2-request@nrao.edu with the following information: your name and mailing address, email address and preferred computer architecture. There is no cost.

AIPS++ is supported by Sun Solaris, Linux RedHat 5.1, 5.2, 6.0, and SuSE 6.0. When requesting a CD-ROM, please specify which version you would prefer. AIPS++ does not currently support Windows NT, Windows 95, nor Windows 98.
