Obtaining AIPS++
Note: The AIPS++ binary installations povide a minimal subset of the AIPS++ data repository. This version is not appropriate for development. Support for users not affilated with the ALMA project may be delayed or deferred.

Download:

To download, point your browser to or ftp directly from ftp://ftp.cv.nrao.edu/aips++/end-user-rpms/. Here you will find directories containing RPM (RedHat Package Manager) files for each of the supported OSs -- RedHat 7.2, RedHat 9, SuSE 8.2, and Vine 2.6r3. These directories include the minimal set of RPMs required to run AIPS++ for each OS.

The full set of OS specific application and OS independent data repository RPMs are also available via FTP.

Should you wish to update your installation, new RPMs will be built following each stable release. They can be downloaded from the location listed above.

For Solaris users, existing AIPS++ installations can be brought up to date using the 'aupdate' command.

Install:

Installation requires root privileges. To install the AIPS++ data reduction package, login as root, change to the directory where you stored the downloaded RPM files, and execute the following commands from the UNIX command line:
  1. Install required PGPLOT libraries

    rpm -ivh pgplot*

  2. Install other needed utilities

    rpm -ivh cfitsio* rpfits*

  3. Install AIPS++

    rpm -ivh aips++*

  4. Install the Compiler Libraries (these install into /usr/gcc/)

    rpm -ivh g77share* gccshare* gppshare*

Note: If you have a prior installation of AIPS++, some RPMs may fail to install. This will be indicated by a message like:

file "FILENAME" from install of "RPMNAME" conflicts with file from package "RPMNAME"

Should this happen, reinstall these RPMs with "rpm -Uvh" rather than "rpm -ivh".

Running AIPS++:

To use AIPS++, your machine needs to have at least 512 MB of memory (RAM) with at least 1 GB of swap space, though more is better. The disk space required varies depending upon the dataset. Having 5 GB should be safe in all cases.

The above steps install the AIPS++ DRP into /usr/bin, /usr/lib, /usr/lib/aips++, and /usr/gcc. AIPS++ can be started simply by typing "aips++" at the UNIX command prompt.
