Obtaining AIPS++

The released version of AIPS++ is available only as a binary distribution (but with source code) on one CDROM. AIPS++ supports use of Solaris on Sparc architectures, and Linux RedHat 7.*, and SuSE 6.2 on Intel architectures. If you wish to receive a copy of the latest CDROM, please email aips2-request@nrao.edu with the following information: your name and mailing address, email address and preferred computer architecture. There is no charge.

AIPS++ does not currently support any version of Windows or any Apple computers.

Alternatively, the ISO image of the CDROM may be downloaded from via the internet. The image is 582MB,so this download requires a reasonably good network connection. For Information on ISO images and creating CDs we suggest visiting xcdroast.

These images are presently available in the USA at ftp://ftp.ncsa.uiuc.edu/aips/ and in Australia at ftp://ftp.atnf.csiro.au/pub/software/aips++/.
