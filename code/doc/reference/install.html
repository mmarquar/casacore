Obtaining AIPS++

Note: The following provides a binary installation of AIPS++ with only the required parts of the data repository. This version is available only for running the package and is not appropriate for development.

Download:

RPM (RedHat Package Manager) files (containing a binary installation of the AIPS++ software and associated data and library files) are available at:

ftp://ftp.cv.nrao.edu/aips++/TST1

Should you wish to update your installation, new rpms will be built following each stable and can be downloaded from this location.

Currently supported OSs are RH7.2, RH9.0, SuSE8.2 [and Vine2.6r3].

  • rh9: source rpms for Red Hat 9.0
  • suse: source rpms for SuSE 8.2
  • rh7.2: source rpms for Red Hat 7.2
  • vine: source rpms for Vine Linux 2.6r3 (beta)

Note: For users of Solaris, a previous installation can be brought up to date using the 'aupdate' command.

To download, use your browser to select all of the rpms in the appropriate OS directory and download them to your computer.

Alternatively, you can follow the command line download below:

  1. Save the OS appropriate file into the directory you want to store the RPMs. This is an ftp command file to retrieve the appropriate RPMs.

    suckrpm_rh9
    suckrpm_suse
    suckrpm_vine

  2. copy the following line into the file .netrc in your home directory (create the file if you don't have one):

    default login anonymous password user@site

  3. Execute the following command to download the relevant files into your current directory.

    ftp ftp.cv.nrao.edu < suckrpm_rh9 (or vine)

    Install:

    To install the AIPS++ data reduction package, execute the following linux rpm commands (requires root privilege) where the RPMs are located. Note: If you already have a prior installation of the AIPS++ package through this rpm install, then do "rpm -Uvh" to update rather than install; this will be indicated by a message like the following:

    file "FILENAME" from install of "RPMNAME" conflicts with file from package "RPMNAME".

    The rpms are [205 MB].

    The required memory is 512 MB with at least 1 GB of swap space, though more is better. The disk space required varies depending upon the dataset. Having 5 GB should be safe in all cases.

    To install AIPS++, cut and paste (or type) the following lines to a unix command line prompt; you will need 'root' permission.

    1. Install AIPS++

      rpm -ivh aips++*

    2. Install required PGPLOT libraries

      rpm -ivh pgplot*

    3. Install other needed utilities

      rpm -ivh cfitsio* rpfits*

    Running AIPS++:

    This will install the AIPS++ DRP into /usr/lib/aips++. AIPS++ can be started simply by typing:

    > aips++

    at your unix command prompt.
