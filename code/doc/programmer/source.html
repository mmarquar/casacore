Access to AIPS++ source code

AIPS++ is an open project. We write most of the code ourselves, and it is copyrighted under the GNU Copyright. Some parts of the code (e.g., mathematical routines) are in the public domain or otherwise fully redistributable.

The most interesting directories containing code are:

  • code: all AIPS++ source code in "expanded" format, i.e. not in a tar file.
  • master: The current working set of RCS version files for the AIPS++ source code. These contain revision information for this and the preceeding release.
  • archive: The archived set of RCS version files for the AIPS++ source code. These contain all revision information from the beginning of the project to the start of the current release.
  • code_ftp: Compressed tar files containing the source code. However, don't attempt an installation before consulting the experts at aips2-sitemgr@nrao.edu!
  • import: utilities and miscellaneous software packages needed for AIPS++ but which are not part of the main AIPS++ code.
  • master_ftp: compressed tar files used in the automatic update of AIPS++ consortium sites.

You should not need anything outside the above directories. But the entire AIPS++ directory structure is accessible, starting from the top-level directory.

The docs directory contains formatted versions of all the project documentation. If your WWW browser is configured properly, PostScript files should appear in the Ghostview PostScript viewer.
