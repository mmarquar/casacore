43 // Vector of Euler rotation angles

51 // <prerequisite>
52 // <li> <linkto class=Vector>Vector</linkto> class
53 // <li> <linkto class=Quantum>Quantum</linkto> class for units
54 // <li> <linkto class=RotMatrix>RotMatrix</linkto> class for usage
55 // </prerequisite>
56 //
57 // <etymology>
58 // Euler angles describe the rotation of a coordinate system
59 // </etymology>
60 //
61 // <synopsis>
62 // The Euler class is a vector of three angles, together with a vector of
63 // three signed integers. The angles describe the rotation around an axis of a
64 // coordinate system, the integers the actual axis around which to rotate.
65 // The integer can be 0 (do not use this angle) or 1,2,3 to indicate the
66 // axis.
67 // Given angles (a1,a2,a3) and axes (i1,i2,i3), the actual rotation matrix
68 // constructed will be:<br>
69 // R = R<sub>i3</sub>(a3).R<sub>i2</sub>(a2).R<sub>i1</sub>(a1) <br>
70 // It has the following constructors:
71 // <ul>
72 // <li> Euler() creates a zero filled vector of length 3. Axes: (1,2,3)
73 // <li> Euler(Euler) creates a copy
74 // <li> Euler(Double, uInt, Double=0, uInt=0, Double=0, uInt=0) creates an
75 // Euler with specified values
76 // <li> Euler(Double, Double=0, Double=0) creates an Euler with (1,2,3)
77 // <li> Euler(Quantity, uInt, Quantity=0, uInt=0, Quantity=0, uInt=0) creates
78 // an Euler with specified values
79 // <li> Euler(Quantity, Quantity=0, Quantity=0) creates an Euler with
80 // interpretation of angle units in the Quantities
81 // <li> Euler(<src>Quantum<Vector<Double> ></src>) creates a zero expanded
82 // Euler from at most the first three elements of Quantity
83 // vector; with (1,2,3)
84 // <li> Euler(<src>Quantum<Vector<Double> >, Vector<uInt></src>) creates a
85 // zero expanded Euler with given values
86 // </ul>
87 // It has a unary minus operator, which reverses the sign and order of the
88 // three angles, and the order of the axes, to produce the Euler angles
89 // for a rotation with opposite signs, so that <src>RotMatrix(-Euler)</src>
90 // will generate the inverse rotation matrix as compared with
91 // <src>RotMatrix(Euler)</src>.<br>
92 // getAngle() functions return the Euler angles as a Quantum vector.<br>
93 // Eulers have addition and subtraction (on the angles). Note that this
94 // produces the correct angles for a combined rotation only if the
95 // axes are identical.<br>
96 // A (which) operator returns the indicated angle. Set/get functions
97 // manipulate the axes.
98 // </synopsis>
99 //
100 // <example>
101 // <srcblock>
102 // Quantity angle(25,"deg"); // 25 degrees
103 // Euler eul(angle.get().getValue(),2); // rotate over axis 2 (radians)
104 // RotMatrix rot(eul); // generates rotation matrix
105 // </srcblock>
106 // </example>
107 //
108 // <motivation>
109 // To use generated precession and nutation results
110 // </motivation>
111 //
112 // <todo asof="1995/09/04">
113 // </todo>
